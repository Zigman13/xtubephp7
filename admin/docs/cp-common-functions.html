Control Panel Common Functions
Table of Contents
Item Functions
Item functions are the icons that appear for each item in the search results, and will be applied only to the corresponding item.


A few of the item functions are very common and will be seen for most items in the control panel. Those are:
    - Edit the item. When you click this icon, a dialog will appear which will have a form that is pre-filled with the item's current settings which you can modify as needed.
    - E-mail the item. When you click this icon, a dialog will appear where you can enter the subject and message of the e-mail that you would like to send.
    - Delete the item. When you click this icon, you will be asked to confirm that you want to delete the item. If you confirm the deletion, the item will be permanently removed from the database. This cannot be undone!
Toolbar Functions
Icons on the toolbar will be applied to multiple items at once. The items to which the function is applied will depend on whether or not you have checked any of the item checkboxes. If any checkboxes are checked, only those checked items will be affected. If no checkboxes are checked, the function will be applied to all items that match the current search you have performed.


A few of the toolbar functions are very common and will be seen for most items in the control panel. Those are:
    - Show Saved Searches tab
    - Add a new item. When you click on this icon, a dialog will appear which will have a form that you will need to fill in with the values you want to use for the new item.
    - Edit the checked or mathing items. When you click on this icon, the Bulk Editing dialog will appear.
    - E-mail the checked or matching items. When you click this icon, a dialog will appear where you can enter the subject and message of the e-mail that you would like to send. The same message will be sent to each item, however you can use template values to customize the exact contents of the message.
    - Delete the checked or matching items. When you click this icon, you will be asked to confirm that you want to delete the checked or matching items. If you confirm the deletion, the items will be permanently removed from the database. This cannot be undone!
    - This toolbar icon does not have any affect on the items you are viewing, and will simply take you to the documentation for the item type you are viewing.
