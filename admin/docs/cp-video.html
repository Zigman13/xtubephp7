Managing Videos
Managing videos is done by accessing the Videos > Search Videos function in the control panel menu. From this interface you can add new videos and search, edit, or delete the existing videos. For an overview of searching, adding, editing, and deleting items, please see the Control Panel Basics section of this software manual.
Videos & Clips
In TubeX there is an important distinction between Videos and Clips. When talking about a Video, that is referencing a complete video object which contains the video title, description, tags, category, and clips.

A Clip, on the other hand, is an individual video file or embed code that is associated with a video. Each video can have multiple clips, but each clip can only be associated with a single video.
Video Sources
Videos can be added through the control panel from a number of different sources. Those are:
    Uploads - This option allows you to upload files from your computer. Multiple files can be uploaded, and multiple file types are supported. Click the icon to add more upload fields, or the icon to remove an upload field. The supported file types are:

    Video Files
    Any video files that are uploaded will be used as the clips for the video. Multiple video files can be uploaded for a single video.

    JPEG Image Files
    Any JPEG image files that are uploaded will be resized (if possible) and used as the preview thumbnails for the video if it is not possible for the software to automatically generate thumbnails from the video files that were uploaded.

    Zip Archives
    Zip files are supported on servers where the PHP zip extension is installed. All video files will be extracted from the archive and used as the clips for the video. All JPEG image files will be extracted, resized if possible, and used as the preview thumbnails for the video if it is not possible for the software to automatically generate thumbnails from the video files that were extracted.
    URLs - This option allows you to enter URLs to files hosted on HTTP servers. Multiple URLs can be entered, and multiple file types are supported. Click the icon to add more URL fields, or the icon to remove an URL field. The supported file types are:

    If you select the hotlink option, the software will not be able to determine the video duration so you will need to fill in the Video Duration field!

    Video Files
    You can enter the direct URL to a video file. Multiple video file URLs can be entered for a single video. If you choose the hotlink option, the videos will be loaded from the URL you enter. Otherwise, the videos will be downloaded and served from your server.

    JPEG Image Files
    You can enter the direct URL to a JPEG image file. Multiple image file URLs can be entered for a single video. The JPEG image files will be downloaded from the URL(s), resized if possible, and used as the preview thumbnails for the video if you have selected to hotlink the videos or the software is not able to automatically generate preview thumbnails from the video files.

    Zip Archives
    Zip files are supported on servers where the PHP zip extension is installed. The Zip Archives will be downloaded to your server. All video files will be extracted and used as the clips for the video. All JPEG image files will be extracted, resized if possible, and used as the preview thumbnails for the video if it is not possible for the software to automatically generate thumbnails from the video files that were extracted.
    Gallery - This option allows you to enter the HTTP URL of a TGP gallery that contains videos. A TGP gallery is considered to be a page which contains JPEG images which link directly to video files. The software will locate all of the JPEG images that link to video files, and download the JPEG images to your server. If you select the hotlink option, the videos will be linked from the server hosting the gallery, otherwise they will be downloaded and served from your server.

    If you select the hotlink option, the software will not be able to determine the video duration so you will need to fill in the Video Duration field!

    The JPEG images from the gallery will be resized if possible and used as preview thumbnails for the video when the hotlink option is enabled or if the software is unable to automatically generate thumbnails from the downloaded video files.
    Embed Code - This option allows you to specify the exact <embed> or <object> code to use for the video. This is useful for embedding videos from other tube sites.

    When using this source type, it is necessary to fill in the Video Duration field specifying the length of the video. You can also optionally upload thumbnail preview images (JPEG format) to be used for the video, since the software cannot generate preview thumbnails from embed code.
Some of the video sources have additional settings and options. Those are:
    Hotlink video clips - This will be an option for URLs and Gallery sources. If this box is checked, the video files will be hotlinked from the server where the URL points (for URLs source) or where the gallery is hosted (for Gallery source). Be sure to verify that hotlinking is allowed before you use this option!
    Queue for thumbnail generation - This will be an option for URLs and Gallery sources. Check this box to have the video added to the thumbnail generation queue. Thumbnail generation will download the video file(s) from the remote server and generate preview thumbnails from them. The video files will be deleted after the thumbnails have been generated, and the videos will be hotlinked when viewed on your site.

    This is only valid when hotlinking the video clips

    Queue video for conversion - This will be an option for all sources except Embed Code. Check this box to have the video(s) added to the conversion queue. The conversion queue will download the video file(s) (if necessary), convert them into the format you have specified in the Global Settings interface, and the converted videos will be hosted from your server. Videos that are queued for conversion will not appear on your site until they have been converted.
Database Fields
Videos have the following fields associated with them:
    Title - The title that has been assigned to the video
    Description - The description that has been assigned to the video
    Tags - The tags that have been assigned to the video
    Username - The username of the account that is associated with this video
    Date Added - The date and time the video was added to the database
    Date/Time Recorded - The date the video was recorded
    Location Recorded - The location where the video was recorded
    Category - The category in which this video appears
    Sponsor - The sponsor from which this video comes
    Comments Allowed - Indicates if comments are allowed for this video. Options are as follows:

    No
    Comments are not allowed at all for this video.

    Yes - Add Immediately
    Comments are allowed and will be appear immediately without any type of review

    Yes - Require Approval
    Comments are allowed but must be approved by an administrator before they will appear
    Private - Indicates if this video should be private. Private videos have special URLs and will not be displayed with the other videos on your site. Only users that have the special URL will be able to access the video.
    Featured - Indicates if this video should be a featured video. Featured videos can be displayed in special sections of your site if you wish. This will depend on your template configuration.
    Allow ratings - Indicates if ratings are allowed for this video
    Allow embedding - Not currently supported; will be added in a future update
    Indicates if this video can be embedded in remote sites
Individual Functions
Besides the Common Functions available for most items, videos also have some custom functions that can be run. The icons that appear for each video in the Search Videos interface are described below.
    - View the comments for this video. This will take you to the Search Comments interface which will display only the comments for this video.
    - Approve this video. This will only be available for videos with Pending status. Approving a video will change the video's status to Active and send the email-video-approved.tpl e-mail message to the user who submitted this video (if any)
    - Reject this video. This will only be available for videos with Pending status. Rejecting a video will delete the video from the database permanently and send the email-video-rejected.tpl e-mail message to the user who submitted this video (if any)

    When you reject a video you also have the option to select a rejection reason. There will be a drop-down selection list that you can choose from. To setup these reasons, use the Search Reasons interface.
    - Enable this video. This will only be available for videos with Disabled status. Enabling a video will change it's status back to Active and it will be visible on your site.
    - Disable this video. This will only be available for videos with Active status. Disabling a video will change it's status to Disabled and it will no longer be visible on your site.
    - Display a dialog which will show you the reasons (if any) that users have flagged this video. From that dialog you can delete or disable the video as well as clear out the flag reasons.
    - Display a dialog which will show the reasons (if any) that users have submitted feature requests for this video. From that dialog you can feature or un-feature the video as well as clear out the reasons.

    This icon will be greyed for videos that are not featured, and will appear as it is here for videos that are featured.
    - Add this video to the conversion queue
    - Add this video to the thumbnail generation queue
Bulk Functions
Besides the Common Functions available for most items, videos also have some custom functions that can be run. The icons that appear in the toolbar which can be run on multiple videos at once are covered below.
    - Approve the checked or matching videos. Only videos with Pending status will be affected by this function. Approving a video will change the video's status to Active and send the email-video-approved.tpl e-mail message to the user who submitted this video (if any)
    - Reject the checked or matching videos. Only videos with Pending status will be affected by this function. Rejecting a video will delete the video from the database permanently and send the email-video-rejected.tpl e-mail message to the user who submitted this video (if any)
    - Enable the checked or matching videos. Only videos with Disabled status will be affected by this function. Enabling a video will change it's status back to Active and it will be visible on your site.
    - Disable the checked or matching videos. Only videos with Active status will be affected by this function. Disabling a video will change it's status to Disabled and it will no longer be visible on your site.
    - Feature the checked or matching videos. Only videos that are not currently featured will be affected by this function
    - Un-feature the checked or matching videos. Only videos that are currently featured will be affected by this function
    - Add the checked or matching videos to the conversion queue
    - Add the checked or matching videos to the thumbnail generation queue
Video Status
Videos can have one of 4 values for their status, depending on how they have been processed and what software settings you are using.
    Pending - These are videos that have been uploaded or otherwise added to the database but not yet approved by an administrator. These videos will not appear on your site until they have been approved. You will get Pending videos when you have the Review uploaded videos before displaying on site software setting enabled.
    Queued - These are videos that have been uploaded and are in the queue to be converted. These videos will not appear on your site until they have been converted. You will get Queued videos when you have the Convert uploaded videos software setting enabled.
    Active - These are videos which are currently visible on your site
    Disabled - These are videos that have been disabled by an administrator. Disabled videos will remain in your database, but will not appear on your site. Disabled videos can only be re-activated by an administrator.
