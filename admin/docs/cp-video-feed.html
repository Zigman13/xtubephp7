Managing Video Feeds
Managing video feeds is done by accessing the Videos > Video Feeds > Search Feeds function in the control panel menu. From this interface you can add new video feeds and search, edit, or delete the existing video feeds. For an overview of searching, adding, editing, and deleting items, please see the Control Panel Basics section of this software manual.
Database Fields
Video Feeds have the following fields associated with them:
    Feed Name - A name for the feed, for easier identification
    Feed URL - The full URL to the feed
    Feed Type - The type of feed. The following feed types are supported:

    XML Videos Feed
    This is a feed that meets this specification. These types of feeds are often offered by adult sponsors. A list of sponsors that offer these feeds is maintained in the TubeX forum.

    When videos are imported from this type of feed, the actual video clips will be hotlinked from the remote server so it won't use any of your bandwidth. The preview thumbnail images will be downloaded to your server, resized if possible, and used as the preview images for this video.

    YouTube
    This is a feed that uses the YouTube Data API. Videos imported from a YouTube feed will use the embed code provided by YouTube. This will display the YouTube player, and the video will be loaded from YouTube's servers so it won't use your bandwidth.

    The preview thumbnail images for the video will be downloaded to your server and used as the preview images for imported videos.
    Sponsor - The sponsor associated with this feed. All videos imported from this feed will be assigned this sponsor.
    Category - The default category to use for videos imported from this feed. Videos which cannot be categorized using the Automatic Categorization feature will be assigned to this category.
    Username - The username to set for videos imported from this feed
    Status - The status to set for videos imported from this feed
    Comments Allowed - Indicates if comments are allowed for this video
    Allow ratings - Indicates if ratings are allowed for this video
    Allow embedding - Indicates if this video can be embedded in remote sites
    Make private - Set videos imported from this feed as private
Individual Functions
Besides the Common Functions available for most items, video feeds also have some custom functions that can be run. The icons that appear for each video feed in the Search Feeds interface are described below.
    - Click this to have the software read the feed and import the videos it finds in the feed. The software will keep track of what videos were imported from each feed to prevent duplicates from being imported. Because of this, it will not be too uncommon to receive a message that 0 videos were imported from the feed when running this function.

    Videos can also be imported from feeds using cron. This will allow you to have videos imported from your feeds automatically without any interaction from an administrator.

    - Click this icon to test the feed. This will check that a valid feed URL was provided, and that the feed is a valid feed of the type you selected.
