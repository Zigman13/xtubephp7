Managing Video Comments
Managing video comments is done by accessing the Videos > Video Comments > Search Video Comments function in the control panel menu. From this interface you can add new video comments and search, edit, or delete the existing video comments. For an overview of searching, adding, editing, and deleting items, please see the Control Panel Basics section of this software manual.
Database Fields
Video Comments have the following fields associated with them:
    Username - The username of the account which submitted this comment.
    Date - The date and time the comment was added.
    Status - The status of the comment

    Pending
    The comment is still waiting to be approved by an administrator and will not appear on your site.

    Active
    The comment is active and will appear on your site.
    Comment - The comment itself.
Individual Functions
Besides the Common Functions available for most items, video comments also have some custom functions that can be run. The icons that appear for each comment in the Search Comments interface are described below.
    - View the video associated with this comment. This will take you to the Search Videos interface which will display only the video that is associated with this comment.
    - Approve this comment. This will only be available for comments with Pending status. Approving a comment will change it's status to Active and the comment will then be visible on your site.
    - Reject this comment. This will only be available for comments with Pending status. Rejecting a comment will delete the comment from the database permanently.
Bulk Functions
Besides the Common Functions available for most items, video comments also have some custom functions that can be run. The icons that appear in the toolbar which can be run on multiple comments at once are covered below.
    - Approve the checked or matching comments. Only comments with Pending status will be affected by this function. Approving a comment will change the comments's status to Active and make it visible on your site
    - Reject the checked or matching comments. Only comments with Pending status will be affected by this function. Rejecting a comment will delete the comment from the database permanently.
