Managing User Levels
Managing user levels is done by accessing the Settings > User Levels > Search User Levels function in the control panel menu. From this interface you can add new user levels and search, edit, or delete the existing user levels. For an overview of searching, adding, editing, and deleting items, please see the Control Panel Basics section of this software manual.
Database Fields
User Levels have the following fields associated with them:
    Name - The friendly name assigned to this user level
    View Limit - The number of videos that user accounts assigned to this user level are allowed to view each day
    Bandwidth Limit - The amount of bandwidth that user accounts assigned to this user level are allowed to consume each day. This can either be a numerical value (interpreted as bytes) or a numerical value followed by one of these suffixes: KB, MB, GB

    Bandwidth limits apply only to locally hosted videos. Videos that use embed code or are hotlinked will not count toward the bandwidth limit.

    Use for guests - Check this box to use this user level for all guests (users viewing your site that are not logged in). Only one user level can have this box checked.
    Use as default - Check this box to use this user level for all newly created user accounts. Only one user level can have this box checked.
