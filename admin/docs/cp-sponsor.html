Managing Sponsors
Managing sponsors is done by accessing the Sponsors > Search Sponsors function in the control panel menu. From this interface you can add new sponsors and search, edit, or delete the existing sponsors. For an overview of searching, adding, editing, and deleting items, please see the Control Panel Basics section of this software manual.
Database Fields
Sponsors have the following fields associated with them:
    Sponsor Name - The sponsor's name
    Sponsor URL - The sponsor's primary URL
    2257 URL - The URL to the sponsor's 2257 record keeping information page (useful for adult sites)
