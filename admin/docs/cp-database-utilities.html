Database Utilities
Table of Contents
Database Backup
This function will generate a complete dump of the TubeX MySQL database tables so they can be restored in an emergency or if moving your installation to a new server. To use the database optimize feature access the Database > Utilities function from the control panel menu. You will find a section titled Database Backup with details on where the backup file will be generated. Simply enter a filename (a .sql file extension is recommended) and press the Run Database Backup button to generate the backup.

After the backup is generated, you may wish to download this file to your local computer for safe keeping. The directory in which the backup file is generated will be indicated, so you can access that directory to download the file.
Database Restore
This function can restore your MySQL database tables to the state they were in at the time the backup file was generated. Only backup files generated by the TubeX Database Backup function can be used. To use this function you will first need to place a backup file in the data directory of your TubeX installation. The backup file should have a .sql file extension.

Once the backup file is in the data directory, access the Database > Utilities function from the control panel menu. You will find a section titled Database Restore which will contain a list of the .sql files in the data directory. Select the backup file you would like to restore from and press the Run Database Restore button to restore that data into your MySQL database.
Database Optimize
This function allows you to "cleanup" the MySQL database tables that TubeX uses so they are optimized for both speed and disk usage. In general, you will not need to run this command more than once every few weeks, as MySQL is quite efficient in maintaining it's data files. The only time you may need to run it more often is if you delete a large number of videos or other items from your database.

To use the database optimize feature access the Database > Utilities function from the control panel menu. This will load the database utilities page where you fill find a button labeled Run Database Optimize Now. Simply click this button to run the optimization operation.
Database Repair
In some cases, usually after your server recovers from a serious error, your MySQL database may become corrupted. When this happens you will get error messages from TubeX indicating that there is a problem with one or more of your database tables. In most circumstances MySQL is able to repair damaged tables with it's built in functions. TubeX allows you to run these built in functions through the Database > Utilities interface. If you ever get these types of error messages, simply login to the control panel and use the Database Repair function.

To use the database repair feature access the Database > Utilities function from the control panel menu. This will load the database utilities page where you fill find a button labeled Run Database Repair Now. Simply click this button to run the repair operation.
Raw Database Query
This function allows you to run a raw SQL query against the MySQL database that is storing the TubeX data. This feature is intended for advanced users who have experience working with SQL queries and understand the impact they will have on the database. In most cases you will not need to use this feature, unless you have been specifically instructed to do so for tech support purposes.
