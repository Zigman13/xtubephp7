Managing Reasons
Managing reasons is done by accessing the Settings > Reasons > Search Reasons function in the control panel menu. From this interface you can add new reasons and search, edit, or delete the existing reasons. For an overview of searching, adding, editing, and deleting items, please see the Control Panel Basics section of this software manual.
Database Fields
Reasons have the following fields associated with them:
    Type - The type of reason. These include:

    Reject User
    Reasons why a user account might be rejected. You will be able to select from these when processing new accounts.

    Reject Video
    Reasons why a video submission might be rejected. You will be able to select from these when processing new video uploads.

    Flag
    Reasons why a video should be flagged. This will be displayed for end-users to select from on the video-watch.tpl page.

    Feature
    Reasons why a video should be featured. This will be displayed for end-users to select from on the video-watch.tpl page.
    Short Name - A short overview of the reason for doing something. This will be displayed in the drop-down selection fields.
    Description - A full description of the reason for doing something
