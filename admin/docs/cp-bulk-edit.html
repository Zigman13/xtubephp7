Making Bulk Edits
The Bulk Editing feature of TubeX will allow you to quickly make multiple changes to multiple items with only a few keyboard strokes and mouse clicks. All of the search interfaces offer the ability to bulk edit the items you are searching. When you click on the Edit icon in the toolbar, a dialog window will appear that looks like this:

    Apply To - This will tell you what items the edits will be applied to. Before you open the bulk editing interface, make sure you have selected the items that you want to edit either by checking their checkboxes (selected items) or doing a search (matching items) that returns only the items you want to modify.
    Field - Select the field which you want to modify
    Action - Select the action to apply to the field. Not all actions can be applied to all fields. Below is a complete list of the actions that can be performed.

    Set
    This will set the value of the field you selected to the string that you enter in the Value box. The original contents of the field will be completely overwritten.

    Append
    This will append the string you enter in the Value box to the end of the value of the field you selected.

    Prepend
    This will prepend the string you enter in the Value box to the end of the value of the field you selected.

    Add
    This will add the number you enter in the Value box to the value of the field you selected.

    Subtract
    This will subtract the number you enter in the Value box from the value of the field you selected.

    Increment
    This will increment (increase by 1) the field you selected.

    Decrement
    This will decrement (decrease by 1) the field you selected.

    Replace
    This will replace one value with another in the field you selected. In the Value box you will need to enter two string seprated by a comma. The first string will be the search string, and the second string will be the replacement string.

    Trim
    This will remove whitespace from the beginning and end of the value of the field you selected.

    Clear
    This will empty the field you selected.

    Truncate
    This will truncate the value in the field you selected. In the Value box enter a number which specifies the length that the value should be truncated to.

    Uppercase All
    This will convert all alphabetic characters in the value of the field you selected to upper case.

    Uppercase First
    This will convert the first character in the value of the field you selected to upper case. It will have no affect if the first character is not alphabetic.

    Lowercase All
    This will convert all alphabetic characters in the value of the field you selected to lower case.

    Raw SQL
    This allows you to enter a raw SQL string in the Value box which will be assigned to the value of the field you selected.
    Value - The Value you enter will depend on the action:

    Set
    Enter the exact string that you want the field to be set to

    Append
    Enter the string that you want appended to the value of the field

    Prepend
    Enter the string that you want prepended to the value of the field

    Add
    Enter a numeric value that you want added to the value of the field

    Subtract
    Enter a numeric value that you want subtracted from the value of the field

    Increment
    LEAVE BLANK

    Decrement
    LEAVE BLANK

    Replace
    Enter two string seprated by a comma. The first string will be the search string, and the second string will be the replacement string.

    Trim
    LEAVE BLANK

    Clear
    LEAVE BLANK

    Truncate
    Enter a numeric value which is the length that you want the value of the field truncated to

    Uppercase All
    LEAVE BLANK

    Lowercase First
    LEAVE BLANK

    Lowercase All
    LEAVE BLANK

    Raw SQL
    Enter a valid SQL string that can appear on the right side of an assignment operator
