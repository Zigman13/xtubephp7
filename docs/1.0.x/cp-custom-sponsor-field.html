Managing Custom Sponsor Fields

Managing custom sponsor fields is done by accessing the Custom Sponsor Fields > Search Custom Sponsor Fields function in the control panel menu. From this interface you can add new custom sponsor fields or search, edit, and delete the existing custom sponsor fields. For an overview of searching, adding, editing, and deleting items, please see the General Control Panel Overview section of this software manual along with the associated screencast.

Custom Sponsor Field Fields

Custom Sponsor Fields have the following fields associated with them:
    Field ID The ID number assigned to the field. ID numbers are generated in sequential order automatically by the software.
    Name The name to use for the database field
    Field Type The form field type to use for this custom input field
    Label The label to display for this field
    Tag Attributes Extra HTML attributes to use in the HTML tag for this field
    Options A comma separated list of values to use for the options of a select field
    Validators The validators to use for this field
