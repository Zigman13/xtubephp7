Managing Blacklist Items
Managing blacklist items is done by accessing the Settings > Blacklist > Search Blacklist Items function in the control panel menu. From this interface you can add new blacklist items and search, edit, or delete the existing blacklist items. For an overview of searching, adding, editing, and deleting items, please see the Control Panel Basics section of this software manual.
Database Fields
Blacklist Items have the following fields associated with them:
    Type - The type of blacklist item. The avaialble types are:

    Submitter IP
    This will search the IP address of the user when they are creating or editing an account, uploading a video, or submitting a comment

    E-mail Address
    This will search the e-mail address of the user when they are creating or editing their account

    Domain/URL
    This will search the Website URL value submitted by a user when editing their account

    Domain IP
    NOT CURRENTLY USED

    Word
    This will search for words that have been submitted for the following:

      Comments
      The comment text

      Videos
      Title, description, tags, and location recorded

      Users
      Username, name, about me, hometown, current city, current country, occupations, companies, schools, hobbies, movies, music, and books

    HTML
    NOT CURRENTLY USED

    HTTP Header
    NOT CURRENTLY USED

    DNS Server
    NOT CURRENTLY USED
    Regular expression - Check this box if the value is a regular expression. Regular expressions are more complex than standard searches that simply look for the item as-is, but they add a great amount of flexibility. In TubeX the Perl Compatible Regular Expressions format is used for regular expressions. If you are not familiar with regular expressions, please see the Pattern Syntax portion of the PHP manual for additional information. If you check the Regular Expression box on the blacklist add/edit form, the item(s) you enter will be treated as regular expressions and used to match against the values that are submitted by the user.

    If the values you enter are not regular expressions, be sure to leave that box unchecked. This will cause a normal search to occur which will match the item you enter anywhere within the field that is being searched. For example, if you enter the word "car" as a blacklisted word and someone submits a description of "Carpool to work to save gas", this will match the "car" blacklist word item since it appears in the description. Note that the search is case-insensitive and will match not only exact words, but also portions of words.
    Value - The value you want to add to the blacklist.
    Reason - Enter a reason why you are blacklisting this item, if you wish. This will help you keep track of why a specific item has been blacklisted.
