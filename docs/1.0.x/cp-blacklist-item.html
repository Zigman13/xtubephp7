Managing Blacklist Items

Managing blacklist items is done by accessing the Blacklist Items > Search Blacklist Items function in the control panel menu. From this interface you can add new blacklist items or search, edit, and delete the existing blacklist items. For an overview of searching, adding, editing, and deleting items, please see the General Control Panel Overview section of this software manual along with the associated screencast.

Blacklist Item Fields

Blacklist Items have the following fields associated with them:
    Blacklist ID The ID number assigned to the blacklist item. ID numbers are generated in sequential order automatically by the software.
    Type The type of blacklist item
    Regular expression Check this box if the value(s) is a regular expression
    Value The value(s) you want to add to the blacklist. You can enter one item per line to add multiple items at once.
    Reason Enter a reason why you are blacklisting this item, if you wish. If this value is provided, it will be displayed to the user.
