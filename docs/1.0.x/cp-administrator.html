Managing Administrators
Managing administrators is done by accessing the Settings > Administrators > Search Administrators function in the control panel menu. From this interface you can add new administrators and search, edit, or delete the existing administrators. For an overview of searching, adding, editing, and deleting items, please see the Control Panel Basics section of this software manual.
Database Fields
Administrators have the following fields associated with them:
    Username - The administrator account's username
    Password - The administrator account's password
    E-mail Address - The administrator's e-mail address
    Name - The administrator's name
    Account Type - The type of administrator account.

    Superuser
    Superuser accounts will be able to access all of the control panel functions without restriction.

    Editor
    Editor accounts will only be able to access the functions which they have the privilege to access. You will be able to select any of the available privileges for each editor type account.
    Privileges - The administrator account privileges
